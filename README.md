## Solutech Coding Challange (backend-api)

## Install the dependencies
```bash
composer install
```

## Set Up database

## Migrate
```bash
php artisan migrate
```

## Create Personal Passport Key for use on localhost
```bash
php artisan passport:client --personal
When prompted for a name, type in authToken 
```

## Start Server
```bash
php artisan serve
```
